# LutraOS
A Linux Distribution based on Debian KDE with Focus on European Souveranity, Actuality and Privacy. Work in Progress.  

<img width="1408" height="768" alt="1784788825077" src="https://github.com/user-attachments/assets/b4946ed9-92c5-417d-85da-25d24ddf76ec" />

What we do otherwise then Debian:
- Kernel 6.17 from Debian Backports
- Custom KDE Configuration and unique Design
- Firefox ESR was replaced with the official regular Firefox from the Mozilla DEB repository.
- In Future: Newer Nvidia Drivers through official Nvidia Repository, Installation of external Software will be possible through a  own little Software Store
